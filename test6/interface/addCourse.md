# 接口：addCourse  [返回](../README.md)
用例： [添加课程](../case/addCourse.md)

- 权限：
    老师：登录后可以添加课程。

- 功能：
    开设课程。

- API请求地址：
   接口基本地址/v1/api/addCourse/

- 请求方式 ：
    POST
    
- 请求实例 ：
        {
            "teacher_id":"10001",
            "name":"IS",
            "term":"2019-3",
        }

- 请求参数说明:
    |参数名称|说明|
    |:---------:|:--------------------------------------------------------|      
    |teacher_id|教师工号|
    |name|课程名称|
    |term|开课学期，由系统自动根据时间生成|

- 返回实例：

        {
            "status": true,
            "info": null,
            "course_id":"1000001"
       
        }

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
  |course_id|返回系统自增生成的课程ID用于下一步添加评分细则|
